# APEX-1 Mini Dataset Examples

These files are tiny educational samples. They are not real training datasets.
They exist so learners can understand the expected JSONL shapes before using
larger corpora.

| File | Purpose |
|---|---|
| `tiny_text.jsonl` | Plain text pretraining-style examples |
| `tiny_sft.jsonl` | Instruction/chat supervised fine-tuning examples |
| `tiny_preference.jsonl` | Chosen/rejected preference pairs for DPO/GRPO lessons |
| `tiny_vision.jsonl` | Vision instruction examples using `<|img|>` |
