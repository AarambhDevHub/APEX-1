## Architecture Design

The APEX-1 architecture was designed by the Aarambh Dev Hub team, synthesizing
the best innovations from frontier AI labs into a single coherent design.

## Acknowledgments

APEX-1 draws architectural inspiration from the following models and organizations:

- **Claude** (Anthropic) — Constitutional AI alignment, reasoning approach
- **GPT-4.5** (OpenAI) — Process Reward Models, scaling insights
- **DeepSeek-V3/R1** (DeepSeek) — Multi-Head Latent Attention, GRPO, auxiliary-loss-free load balancing
- **Qwen3** (Alibaba) — Large vocabulary (151k tokens), multilingual coverage
- **Gemma 4** (Google) — Interleaved local/global attention pattern
- **GLM-4** (Zhipu AI) — Prefix bidirectional attention
- **KIMI** (Moonshot AI) — YaRN context extension, long-context handling
- **MiniMax** (MiniMax) — Efficient MoE design
- **Llama 3** (Meta) — GQA + sliding window attention, SwiGLU FFN
